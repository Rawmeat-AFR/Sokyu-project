# 1.**概要**
軍事部門とは、陸軍、宙軍に次ぐ第三の軍事組織として専門的な分野を担当する組織である。

---
# 2.**一覧**

1. 監察警務軍：帝國内の治安維持と監察が主な任務である
2. 広域機動空挺軍：機動力を特徴とし、有事の際に迅速に任務を遂行する。
3. 特殊作戦軍：帝國軍最高峰の練度を誇り、高度な任務を行う。
4. 電子戦略軍：サイバー戦や情報戦、帝國全域のネットワーク防衛など、電子戦を専門とする。
5. 軌道防衛軍：帝國領内の惑星やコロニー、政府施設などの防衛を行う。
