# 　　執行官

## 　概要

執行官とは、監察官が発した命令を遂行する将校である。

## 　階級

| 階級名 |
| --- |
| 執行総監 |
| 統括執行官 |
| 上級執行官 |
| 執行官 |

## 　権限

基本的に執行官単独での裁定はできず、監察官の裁定を実行する役割である。

## 　部隊

執行官は基本的に監察官一人に対して二人が警護を行う。

---

# 　　警察活動

## 　概要

執行官は各地の要請に基づいて警察活動を支援することができる。
