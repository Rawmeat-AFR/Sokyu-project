# 　　概要

監務庁とは、監査警務軍において軍内部の監査を担う組織である。

---

# 　　監察官

監務庁に所属する将校であり、階級は最低でも少尉以上である。

## 　階級

| 階級名 |
| --- |
| 監察総監 |
| 統括監察官 |
| 特別監察官 |
| 上級監察官 |
| 監察官 |

## 　権限

監察官は階級を問わず帝國軍所属すべての軍人に対し監察を行う権限を有し、いかなる階級者であろうと捜査を阻害することは許されない。

---

# 　　裁定

## 　命令

監察官の命令は監察業務に関する場合は全ての階級者に対して作用する。

## 　執行

監察官が発した命令の執行は執行官によって行われる。

## 　撤回

監察官の裁定は上位の監察官の裁定のみで覆される。
